# Inicar

- Clonar repositorio

``` sh
  git clone https://github.com/Miguel-r-dev/api-fb-clone-mitm.git
```

- Instalar dependencias

```sh
  npm install 
```

- Iniciar el proyecto

```sh
  nodemon index.js
```
